# 地产拍卖领域知识(依据 GDD §2.1 / §3.1 / §4.2)

## 触发条件

玩家落在无主地产且选择不购买时,该地产**立即**进入英式拍卖(English auction,公开增价)。

## 规则要点(GDD 锁定项)

- 起拍价:地价的 50%;
- 加价步长:10 金;
- 出价顺序:全体未破产玩家轮流出价或弃权;
- 流拍:连续全体弃权 → 地产保持无主;
- 成交:最高出价者立即按出价支付并获得地契(所有权转移走 property_economy 的支付与地契路径)。

## 设计空间(可探索维度)

- 出价时限策略:无时限轮转 / 每轮限时(Phase 1 建议无时限,本地热座);
- 弃权后可否重新入场:标准英式拍卖弃权后仍可再出价 vs 弃权即退出(需收敛选择);
- 流拍判定窗口:一轮全弃权 vs 连续两轮(GDD 写"连续全体弃权",取一轮全弃权即流拍的最小解释,标注 provisional);
- 平局保护:不存在(增价拍卖天然无平局)。

## 表现要求(GDD §3.1 / §4.2)

拍卖面板为屏幕中央弹出层:当前地产名、当前最高价、出价记录、出价按钮、弃权按钮;
HUD 在拍卖期间须持续可见当前最高价(§2.1)。
